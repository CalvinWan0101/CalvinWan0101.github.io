Markdown 語法的筆記

本筆記為課程 Github 免費架站術！輕鬆打造個人品牌 中第三章第一單元所做的小整理

本筆記的表述方式

在方格內為md檔案的文字

斜體則代表呈現出來的文字

內文

一般的表述

我今天吃了一份蛋餅還喝了一杯奶茶

今天我吃了一份蛋餅還喝了一杯奶茶

換行需要中間空一行

//未空一行
我今天吃了一份蛋餅
還喝了一杯奶茶

今天我吃了一份蛋餅還喝了一杯奶茶

//空一行
我今天吃了一份蛋餅

還喝了一杯奶茶

我今天吃了一份蛋餅

還喝了一杯奶茶

標題

字號的大小以 # 的數量來表示，可以從1級至6級

//一級標題 
# 柴犬

柴犬

//三級標題 
### 柴犬

柴犬

//六級標題 
###### 柴犬
柴犬

總結一下來對照( # 越多字越小)

# 柴犬
## 柴犬
### 柴犬
#### 柴犬
##### 柴犬
###### 柴犬

柴犬

柴犬

柴犬

柴犬

柴犬
柴犬

斜體

*柴犬跟柯基是不同的生物*

柴犬跟柯基是不同的生物

粗體

**柴犬跟柯基是不同的生物**

柴犬跟柯基是不同的生物

無序序列

[*] 號後先打一個空格再打出你想要打的文字

* 洗衣服
* 煮泡麵
* 帶錢包
  • 洗衣服
  • 煮泡麵
  • 帶錢包

有序序列

先打 1 再打 . 然後空格，最後打你要打的文字

1. 洗衣服
2. 煮泡麵
3. 帶錢包
  1. 洗衣服
  2. 煮泡麵
  3. 帶錢包 可以按照順序也可以全部都打 1 他會幫你自動計算每一項應該是幾
1. 洗衣服
1. 煮泡麵
1. 帶錢包
  1. 洗衣服
  2. 煮泡麵
  3. 帶錢包
